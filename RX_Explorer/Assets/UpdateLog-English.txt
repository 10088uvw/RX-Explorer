###What's new in Ver.7.7.9.0

#####Performance improvement

>#####Reduce the time between finding and restarting an offline auxiliary process

>#####The scheduling algorithm of the auxiliary process scheduler has been improved, and the progress of the task queue will not be hindered when encountering an offline auxiliary process

>#####Introduces a priority queue to the auxiliary process scheduler, improving the responsiveness of user operations and reducing the time when processing a large number of files

#####Bug Fixes

>#####Fixed an issue where file locations could not be opened from the search page

>#####Fixed an issue where delete operations could not be undone

>#####Fixed an issue that an error prompt pops up when clicking the "Quick Access" node
