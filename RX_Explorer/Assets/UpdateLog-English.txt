###What's new in Ver.7.4.1.0

#####Feature update

>#####Added "Security" tab in property window to indicate permissions for each user

>#####Added settings to control the behavior when the context menu pops up

>#####Added rename function for Library and Drive

#####UI update

>#####Redesigned the UI of the disk property window, adding a number of disk-related functions

>#####Redesigned the UI of "Tool" tab in file property window

>#####Redesigned the mechanism for canceling tasks to control tasks executed under administrator privileges

#####Bug Fixes

>#####Fixed an issue that could cause a crash in some cases

>#####Fixed an issue where Windows Terminal might not be properly detected and profiled
