###What's new in Ver.7.6.5.0

#####UI updated

>#####Users will be asked before removing folders that no longer available in the Quick Access

>#####Search page now supports renaming and some new menu items have been added

#####Bug Fixes

>#####Fixed the problem that the unlocking failure dialog is still being pop up after the Bitlocker is unlocked

>#####Fixed an issue where the properties window could crash in some cases

>#####Fixed the issue that after opening multiple tabs at the same time, the tab that stays on the search page may respond to the shortcut keys of other tabs

>#####Fixed an issue that could result in incorrect names when renaming multiple files in an MTP device

>#####Fixed an issue that could cause a crash when renaming a drive with some special symbols

>#####Fixed some other issues that could cause crashes
