###Was ist neu in Ver.7.2.7.0

#####Funktionsupdate

>#####Unterstützung für ARM64 hinzugefügt

>#####Unterstützung für Win11 SDK hinzugefügt

>#####Unterstützung für die Anzeige des Fortschritts in der Taskleiste hinzugefügt

>#####Unterstützung für das Abbrechen von Aufgaben im Status "Vorbereitung" hinzugefügt

#####UI-Update

>#####Verhalten von Quicklook geändert, wenn keine Dateien ausgewählt sind

#####Leistungsverbesserung

>#####Die Leistung der integrierten Suchmaschine wurde in einigen Fällen verbessert

#####Framework-Update

>#####Der Hilfsprozess wurde auf das .Net 6-Framework aktualisiert

#####Fehlerbehebung

>#####Problem behoben, das in einigen Fällen zu einem Absturz in der Win11-Systemumgebung führen kann

>#####Es wurde ein Problem behoben, das den Import von Konfigurationsdateien blockieren kann

>#####Das Problem wurde behoben, dass, wenn der beim Start geöffnete Pfad nicht existiert, die Animation beim Start versehentlich übersprungen wird

>#####Probleme mit der Benutzeroberfläche in "Einstellungen" und "Andere Programme auswählen" behoben

>#####Das Problem, dass das Banner eine transparente Hintergrundfarbe anzeigen kann, wurde behoben

>#####Problem behoben, dass der eingebaute Viewer im "Öffnen mit" angezeigt werden kann, auch wenn der Dateityp nicht unterstützt wird

>#####Das Problem wurde behoben, dass UWP-Apps möglicherweise nicht als Standardprogramm festgelegt wurden

>#####Es wurde ein Problem behoben, bei dem das Standardterminal nicht geändert werden konnte
