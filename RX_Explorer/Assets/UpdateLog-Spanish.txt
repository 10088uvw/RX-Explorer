###Novedades de la versión 7.5.4.0

#####Característica actualizada

>#####Soporte agregado para dispositivos MTP (teléfonos inteligentes, tarjetas SD, etc.)

>>#####La función de monitoreo de cambios de archivos no es compatible con dichos dispositivos y no admite operaciones de archivos simultáneas

>>#####Según el dispositivo, es posible que algunos dispositivos no admitan una o más operaciones, como Mover/Copiar/Eliminar/Renombrar, las operaciones no admitidas fallarán

#####IU actualizada

>#####Se mejoró el estilo de algunas UI en la "Página de inicio"

>#####Las subopciones aparecerán automáticamente al cambiar el modo de interfaz de usuario en la configuración

>#####"Lista de tareas" se ha incorporado como parte del contenido de cada pestaña para evitar oscurecer los botones de algunas funciones clave

>#####El visor de imágenes incorporado ahora usará directamente la imagen original para obtener una vista previa en ausencia de una miniatura

>#####Se mostrará una barra de progreso cuando una carpeta tarde demasiado en cargarse para indicar que aún se está cargando

>#####Clasificación mejorada de nombres de archivos, ahora será consistente con el Explorador de Windows

#####Error corregido

>#####Se corrigieron algunos problemas de la interfaz de usuario en la página de configuración al cambiar de tema

>#####Se solucionó un problema por el cual un dispositivo USB mal conectado podía hacer que la interfaz de usuario se congelara

>#####Se solucionó un problema por el cual el visor incorporado admite archivos .jpg pero no archivos .jpeg

>#####Se solucionó el problema de que parte del contenido de texto en las propiedades de la unidad puede ocultar otros elementos cuando el contenido es demasiado largo

>#####Se solucionó un problema por el cual el tamaño de la ventana actual no se recordaba al cerrar la ventana de propiedades con el botón Cerrar en la esquina superior derecha

>#####Se solucionó el problema de que la extensión del archivo se podía perder al usar la función de cambio de nombre después de que "Mostrar extensión de archivo" está deshabilitado

>#####Se solucionó un problema por el cual la operación de pegar no se podía realizar varias veces después de copiar ciertos tipos de archivos

>#####Se solucionó un problema por el cual el nombre de archivo de algunos archivos podría estar vacío cuando "Mostrar extensión de archivo" está deshabilitado

>#####Se solucionó un problema por el cual el comando "Enviar a" se podía usar accidentalmente en la raíz de la unidad

>#####Se solucionó un problema por el cual un elemento de menú que podría no coincidir con el tipo de archivo actual podía mostrarse en casos excepcionales

>#####Se solucionó un problema por el cual cambiar el nombre de un nodo en el árbol de directorios de archivos que no es la carpeta actual podría no actualizarse correctamente

>#####Se solucionó el problema de que cuando la aplicación predeterminada y la aplicación recomendada son las mismas, puede haber elementos duplicados en el menú emergente del botón "Abrir con"

>#####Se solucionó un problema por el cual el visor de archivos integrado no mostraba el mensaje "Vacío" al navegar a una carpeta vacía

>#####Se solucionó el problema de que la imagen de vista previa en el modo "Imagen de fondo personalizada" puede tener duplicados en algunos casos
