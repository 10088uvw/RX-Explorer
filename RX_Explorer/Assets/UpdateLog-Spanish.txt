#####Actualización de funciones

>#####Agregada "Lista de tareas", ahora las operaciones relacionadas con archivos no se mostrarán en modo de pantalla completa

>#####Refactorizó la lógica del módulo de gestión de dispositivos USB

>#####Retrasa el tiempo de carga de cada elemento de la interfaz principal, y el área contraída no se cargará al inicio

#####Arreglo del fallo

>#####Se solucionó el problema de que los dispositivos USB no se expulsaran

>#####Se solucionó el problema de que la animación de inicio puede fallar en algunos casos después de usar "Abrir última pestaña"

>#####Se solucionó el problema de que el archivo no se descomprime en el formato de codificación predeterminado

>#####Se corrigieron algunos problemas de traducción
